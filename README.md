# CHRONOS - Luksuzni Satovi

Next.js projekat za prodaju luksuznih satova.

## Instalacija

```bash
npm install
```

## Pokretanje

```bash
npm run dev
```

Otvori [http://localhost:3000](http://localhost:3000)

## Struktura projekta

```
src/
├── app/
│   ├── layout.js
│   ├── page.js
│   └── globals.css
└── components/
    ├── Navbar/
    ├── Hero/
    ├── Story/
    ├── Collection/
    ├── Features/
    ├── CTA/
    ├── Contact/
    └── Footer/
```

## Backend (za kasnije)

Napravićeš folder `backend/` u root-u sa:
- server.js
- routes/
- models/
